GITHUB PAGES DEPLOY GUIDE - ROBI PREMIUM CV WEBSITE

Folder er vitore ei file gulo ache:
- index.html
- styles.css
- script.js
- assets/profile-fallback.svg
- .nojekyll

GitHub e publish korar simple way:
1. github.com e login korun.
2. New repository create korun.
3. Repository name din: my-portfolio-cv ba robi-premium-cv
4. Public select korun.
5. Create repository click korun.
6. Add file > Upload files e giye ei ZIP extract kore sob file upload korun.
7. Commit changes click korun.
8. Repository Settings > Pages e jan.
9. Source select korun: Deploy from a branch.
10. Branch select korun: main
11. Folder select korun: /root
12. Save click korun.
13. Kichu khon por live link hobe: https://yourusername.github.io/repository-name/

Important:
- index.html must be root folder e thakte hobe.
- GitHub e ZIP file upload korben na; ZIP extract kore vitore je files ache ogulo upload korben.
- Phone number, email, portfolio link ei version e nai.
